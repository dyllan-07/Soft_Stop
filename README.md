# Softstop

A Chrome extension that gently interrupts you on distraction sites — not with a hard block, but with a painting and a question.

When you visit a site you've flagged, a full-screen artwork slowly fades in. You're asked to take a breath, then: *are you here on purpose?* If yes, it snoozes and lets you through. If no, the painting stays.

## Install

Load as an unpacked extension in Chrome (`chrome://extensions` → Developer mode → Load unpacked → select this folder).

## Setup

1. Click the extension icon and add your distraction sites (e.g. `reddit.com`)
2. That's it

## Settings

- **Time between stops** — how long before it checks in again (1–30 min)
- **Stop on page load** — show immediately, or wait for the snooze interval
- **Painting preference** — Romantic or Impressionist
- **Opening phrase** — customize the first thing you see
